Web Labs > Reactor temperature control - controller manipulates jacket T

A 1st-order, essentially irreversible reaction in a continuous stirred tank reactor (CSTR). The controller varies the jacket T to control reactor T. Reactor volume = 0.1 m3, jacket volume = 0.02 m3, density = 1000 kg/m3, heat capacity = 2.0 kJ/kg/K. Click the Run button to start. Hit the enter key or click out of field after changing inputs. Copy data button displays plot data in popup window.

At default values in manual mode with constant inputs, the system oscillates. Then put into Auto Control.
